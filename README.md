# 📊 CRM Sales Dashboard & Customer Analytics Project

## 🚀 Project Overview
This project demonstrates a complete Customer Relationship Management (CRM) analytics workflow using real-world simulated data (50,000+ records). The goal is to analyze customer data, track sales performance, and generate actionable business insights using Excel, Google Sheets, and Power BI.

---

## 🎯 Objectives
- Manage and analyze customer data
- Track lead generation and conversion rates
- Build MIS reports for business decision-making
- Create interactive dashboards
- Identify key trends and performance metrics

---

## 🛠 Tools & Technologies
- Microsoft Excel
- Google Sheets
- Power BI
- Data Visualization Techniques
- Pivot Tables & Formulas

---

## 📂 Dataset Details
The dataset includes:
- Customer ID
- Lead Source (Facebook, Website, Referral, etc.)
- Sales Status (Won / Lost / Pending)
- Revenue
- Sales Executive
- Inquiry & Follow-up Dates

Total Records: **50,000+**

---

## 📊 Key Features

### 🔹 1. CRM Data Management
- Structured customer database
- Data cleaning and transformation
- Customer lifecycle tracking

### 🔹 2. MIS Reporting
- Daily, Weekly, Monthly reports
- KPI tracking:
  - Total Leads
  - Converted Leads
  - Conversion Rate
  - Revenue

### 🔹 3. Dashboard (Google Sheets / Power BI)
- Lead Source Analysis
- Monthly Sales Trend
- Conversion Funnel
- Sales Executive Performance

### 🔹 4. KPI Cards
- Dynamic KPI cards linked with pivot tables
- Real-time updates using slicers

---

## 📈 Key Insights
- Referral leads have the highest conversion rate
- Facebook generates maximum leads but lower quality
- Certain months show peak revenue trends
- Top-performing sales executives contribute major revenue

---

## 🎯 Business Impact
- Helps in identifying high-quality lead sources
- Improves sales strategy and targeting
- Enhances decision-making using data-driven insights
